# Gradient Multi-Material TPMS Lattice Dataset Generator

**三维梯度多材料混合TPMS晶格结构数据集生成工具**  
(适用于增材制造、轻量化结构设计、生物医学植入物、功能梯度材料研究等方向)

当前日期参考：2026年3月

## 项目概述

本项目用于大规模生成**三维梯度**、**多材料**、**混合TPMS**（Triply Periodic Minimal Surface）的周期性晶格结构数据集，主要服务于：

- 机器学习驱动的微结构性能预测
- 功能梯度材料（FGM）的正向/逆向设计
- 多相材料复合晶格的力学性能均匀化研究
- 生物医学植入物、航空航天轻量化构件等领域的结构-性能映射建立

核心特点：

- 支持 **7种经典TPMS** 的任意连续权重混合
- 支持 **多材料**（两种材料任意体视配比）
- 支持 **空间连续梯度**（单元尺寸 L、壁厚/相对密度通过水平集常数 t 控制）
- 5种梯度函数类型 × 多种梯度方向
- 使用三维周期边界均匀化（FFT-free，直接有限元均匀化）计算等效刚度张量
- 生成的数据直接以体素场 + 参数场形式保存，便于后续深度学习输入

## 数据集结构（每个样本）

每个样本保存为 `.mat` 文件，包含两个主要 4D 数组：

| 字段名        | 维度                        | 内容说明                                                                 |
|---------------|-----------------------------|--------------------------------------------------------------------------|
| `param_field` | [nx, ny, nz, 2×n_topo + 1] | 第4维通道：<br>1~7：混合权重 w（全局常数）<br>8~14：水平集偏移 t（全局常数）<br>15：局部相对尺寸 L（空间变化） |
| `perf_field`  | [nx, ny, nz, 9]             | 局部等效刚度张量 Voigt 表示的 9 个独立分量（单位：GPa）<br>C11, C22, C33, C12, C13, C23, C44, C55, C66 |

## 支持的TPMS类型（2026版）

| 序号 | 拓扑名称       | 代表性特征                     | 典型相对密度范围 |
|------|----------------|--------------------------------|------------------|
| 1    | Neovius        | 高各向同性，较高刚度           | 中～高           |
| 2    | Schoen IWP     | 良好的剪切性能                 | 中               |
| 3    | Schwartz P     | 经典、各向同性较好             | 中～高           |
| 4    | Gyroid         | 优异的渗透性、曲率均匀         | 中～低           |
| 5    | Diamond        | 高刚度、较高相对密度           | 高               |
| 6    | Lidinoid       | 复杂拓扑，较高各向异性         | 中               |
| 7    | Fischer Koch S | 新兴拓扑，特殊力学响应         | 中～低           |

## 支持的梯度类型

| 类型         | 描述                               | 主要控制参数     |
|--------------|------------------------------------|------------------|
| linear       | 线性梯度                           | —                |
| quadratic    | 二次函数                           | —                |
| sigmoid      | S型过渡（陡峭可调）                | beta（2~10）     |
| radial       | 从中心向外径向增加                 | —                |
| multi_axial  | x·y·z 多轴耦合非线性               | —                |

## 主要依赖文件说明

## 快速开始

1. 准备 MATLAB 并行计算工具箱（推荐 28+ 核）
2. 调整 `generate_final_gradient_dataset.m` 中的参数：

```matlab
N_samples      = 5000;               % 目标样本数
grid_size      = [48,48,48];         % 体素分辨率
materials      = {你的材料对列表};
topology_types = {你的TPMS列表};
grad_types     = {'linear','quadratic','sigmoid','radial','multi_axial'};
grad_directions = {'x','y','z','coupled','radial'};
L_range        = [0.5, 10.0];        % 相对/绝对尺寸范围 (mm)
t_range        = [-1.5, 1.5];        % 水平集偏移范围
